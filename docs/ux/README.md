# UX

> Estado: vigente
> Audiencia: producto, diseno, desarrollo, agentes
> Fuente de verdad: complementaria
> Ultima revision: 2026-04-03

## Objetivo

`ux/` documenta la experiencia visible del sistema:

- pasos que realiza el usuario;
- estados de pantalla;
- mensajes y validaciones;
- criterios de claridad, conversion y confianza.

No reemplaza a `contracts/` ni a `modulos/`. Los complementa.
